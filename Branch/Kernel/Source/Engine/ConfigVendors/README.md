# Configuration Files

Json Example:
```json
{
    
    "<Node Name1>": {
        "function": {"name":"<Function name>","Parameter_name1":"Parameter_value1","Parameter_name2":"Parameter_value2"}
       
    },
    "<Node Name2>": {
        "function": {"name":"<Function name>","Parameter_name":"Parameter_value"}
       
    }

}
```
***important***
- Only one function in each node is allowed.
